Informações sobre o experimento

Funcionamento Geral

Os dados de temperatura e umidade relativa do ar são medidos pelos sensor DHT11 ao passo que os indicadores de qualidade do ar são medidos pelo sensor MQ-135. Estes dados são processados através do Arduino Uno Rev 2 e são enviados para um servidor de internet das coisas. Por fim os dados são mostrados aqui no site.
